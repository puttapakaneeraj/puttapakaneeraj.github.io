# **App Name**: Devfolio

## Core Features:

- Profile Showcase: Display the developer's professional summary, profile photo, and key skills.
- Animated Timeline: Present the developer's professional experience using an animated timeline or card-based layout, highlighting roles, companies, and responsibilities.
- Skills Visualization: Display technical and soft skills with animated progress bars or interactive icons, categorized for easy viewing.
- Project Portfolio: Showcase featured projects with descriptions, tech stacks, screenshots, and live/demo links.
- Contact Form: Implement a contact form with email integration and social media links, including interactive hover animations.
- AI-Powered Resume Tool: The page utilizes a tool which leverages a Large Language Model (LLM) to perform automatic text summarization to condense a resume into a concise version.
- Interactive Resume Download: Offer a downloadable resume with a visually engaging button.

## Style Guidelines:

- Primary color: Deep blue (#223142) for a professional, tech-focused aesthetic.
- Background color: Light gray (#F0F4F8) for a clean and modern look.
- Accent color: Teal (#26A69A) to highlight interactive elements and add a vibrant touch.
- Body and headline font: 'Inter' sans-serif for a modern, neutral and clean presentation.
- Use minimalistic icons for skills and social media links.
- Clean, responsive grid layout with well-defined sections.
- Smooth scroll animations, hover effects, and section transitions for an engaging user experience.
